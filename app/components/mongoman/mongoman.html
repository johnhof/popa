Mongoman

A Node module designed to simplify mongoDB schema & validation construction as well as provide utilities to prevent bloat

Getting Started

Mongoman requires Node and NPM. once those are set up, the module can be installed like any node module. 

$ npm install Mongoman

Any time you want access to Monoman, just require it above your mongodb usage. 

var Mongoman = require('Mongoman');

Ready to go? Check out the documentation

A Quick Look

The goal of Mongoman is to prevent bloat in schema & model creation and help make mongo related tasks painless. I recomment using Mongoman alongside mongoose, an awesome node module that makes mondoDB use relatively simple.

Using Mongoman, you can turn this 


    // some bloated code

into this 


    var mongoman = require('mongoman');
    // some non bloated code

More information
